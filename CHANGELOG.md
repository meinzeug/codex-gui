# Changelog

## 0.1.0 - 2026-03-24

Initial public release.

- Added a GTK3 desktop app that embeds a real Codex CLI session through VTE
- Added text prompt injection into the live Codex terminal
- Added live dictation with silence-based auto-send
- Added voice commands for `enter` and `shotscreen`
- Added screenshot capture and local image attachment to Codex
- Added resume-aware Codex session restart support
- Added supermode watchdog support for self-hosted editing
- Added setup automation, start scripts, desktop launchers, and GNOME favorites integration
