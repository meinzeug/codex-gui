# Changelog

## 0.2.0 - 2026-03-24

Second public release.

- Added global prompt library support across all projects
- Added prompt favorites, folder organization, filtering, reload, and resend actions
- Added compact top-level GUI layout with reduced vertical chrome
- Added explicit Codex mode toggles for `--search` and full-access startup
- Added clearer project context and faster in-window project switching workflows
- Added richer voice/provider configuration and microphone metering
- Improved screenshot command handling and more screenshot trigger variants
- Improved supermode coordination between supervisor and main GUI
- Improved desktop launcher integration and dedicated supermode icon

## 0.1.0 - 2026-03-24

Initial public release.

- Added a GTK3 desktop app that embeds a real Codex CLI session through VTE
- Added text prompt injection into the live Codex terminal
- Added live dictation with silence-based auto-send
- Added voice commands for `enter` and `shotscreen`
- Added screenshot capture and local image attachment to Codex
- Added resume-aware Codex session restart support
- Added supermode watchdog support for self-hosted editing
- Added setup automation, start scripts, desktop launchers, and GNOME favorites integration
